🏥 Medical Appointment WebApp

Μία απλή web εφαρμογή για ιατρείο που επιτρέπει στους ασθενείς να κλείνουν ραντεβού online και στους γιατρούς να διαχειρίζονται το πρόγραμμά τους. Η εφαρμογή υλοποιείται με JSP, Java Servlets και PostgreSQL.

✨ Features

Δημιουργία λογαριασμού ασθενή και σύνδεση.

Προβολή διαθέσιμων ημερομηνιών και ωρών.

Κράτηση ραντεβού μέσω φιλικής web διεπαφής.

Διαχείριση ραντεβού από τον γιατρό (έγκριση/ακύρωση).

Αποθήκευση δεδομένων σε PostgreSQL.

Χρήση Servlets για το backend logic και JSP για το UI.


🛠️ Technologies Used

Java Servlets (Jakarta EE / Java EE)

JSP (Java Server Pages)

PostgreSQL

JDBC

HTML / CSS

Tomcat Server


🚀 How to Run

Εισαγωγή του project σε IntelliJ / Eclipse.

Ρύθμιση PostgreSQL και εκτέλεση του SQL schema.

Ενημέρωση των database credentials στο DBConnection.java.

Deployment σε Apache Tomcat.

Άνοιγμα στο browser:
http://localhost:8080/MedicalApp/

📌 Notes

Αυτό το project αποτελεί απλή υλοποίηση εκπαιδευτικού σκοπού.

Μπορεί εύκολα να επεκταθεί με authentication filters, email reminders, admin panel, κ.λπ.
